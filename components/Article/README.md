# Article

The Article component styles the inner elements such as `<img>`, `<ul>`, `<p>`, etc. To make them look as we want.

## Usage

```
<Article>
  <h1>title</h1>
  <p>Lorem ipsum....</p>
  <ul>
    <h2>subtitle</h2>
    <li>option 1</li>
    <li>option 2</li>
    <li>option 3</li>
  </ul>
</Article>
```
